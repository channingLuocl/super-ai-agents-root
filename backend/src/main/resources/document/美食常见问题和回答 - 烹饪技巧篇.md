# 烹饪技巧大全

## 火候掌握

### 旺火（大火）
- 适用：快速翻炒、爆炒、煎炸
- 特点：火力强，升温快，食物表面迅速收紧
- 注意事项：需要不停翻动，避免糊锅

### 中火
- 适用：红烧、焖煮、炖汤
- 特点：火力适中，热量传递均匀
- 注意事项：保持微沸状态即可

### 小火（文火）
- 适用：慢炖、熬汤、保温
- 特点：火力柔和，缓慢释放热量
- 注意事项：防止干烧，注意水位

## 刀工技巧

### 切菜基本刀法
1. **切**：刀垂直上下运动，适用于脆性食材
2. **片**：刀与食材成水平角度，适用于软性食材
3. **剁**：刀快速上下运动，适用于肉类末、蒜泥
4. **削**：刀外向运动，去除食材外皮

### 常见切法
- **丝**：长5厘米，宽0.3厘米的细条
- **片**：厚0.3厘米的薄片
- **块**：2-3厘米见方的块状
- **丁**：1厘米见方的小块
- **末**：0.3厘米以下的碎末

## 调味技巧

### 基础调味顺序
1. **盐**：烹饪开始或结束时加入
2. **酱油**：上色增香，在油热时加入
3. **糖**：提鲜，中和酸味
4. **醋**：去腥解腻，出锅前加入
5. **味精/鸡精**：提鲜，出锅前加入

### 调味黄金法则
- 少量多次：调味要边尝边加
- 突出主味：一种味道为主，其他为辅
- 平衡协调：酸甜苦辣咸要平衡
- 因材施教：不同食材不同调味

## 食材处理

### 肉类腌制
- **目的**：去腥、入味、保持嫩滑
- **常用调料**：料酒、盐、胡椒粉、淀粉、蛋清
- **时间**：15分钟到2小时不等
- **技巧**：
  - 逆纹切、逆纹腌
  - 加少许油可防止粘连
  - 冷藏腌制效果更好

### 蔬菜保鲜
- **绿叶菜**：用保鲜袋包好，放冰箱冷藏
- **根茎类**：阴凉通风处保存
- **菇类**：用纸袋包装，防止潮湿变质
- **鲜肉**：分割成小份，冷冻保存

## 常见问题解决

### 炒菜粘锅
- 锅要烧热再放油
- 油要稍微多放一点
- 食材要沥干水分
- 急火快炒，不要频繁翻动

### 汤太咸
- 加入土豆片煮一下吸附盐分
- 加入米汤或面粉水勾芡减咸
- 加入适量清水稀释

### 鱼肉腥味重
- 用料酒、姜、葱腌制
- 烹饪时加入醋或柠檬
- 出锅前撒少许胡椒粉

### 炒蛋不够嫩
- 蛋液中加少许温水
- 热锅热油，大火快炒
- 蛋液凝固就出锅，不要炒太久

## 烹饪小贴士

1. **焯水**：冷水时下锅，加少许盐和油，保持蔬菜色泽
2. **过油**：油温六成热下锅，保持食材形状
3. **挂糊**：淀粉加水调匀，裹住食材保持水分
4. **勾芡**：淀粉加水徐徐倒入，快速翻匀
5. **明油**：菜品完成后淋入少量热油，增加光泽和香味
